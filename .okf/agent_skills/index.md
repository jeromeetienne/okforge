# agent_skills

The Claude Code skills bundled with okforge, shipped under `dotclaude_folder/`
and copied into an agent folder by the [install](../cli_commands/install.md)
command.

- [okforge](./okforge.md) — maintain the OKF bundle (scaffold / refresh / check).
- [okforge_browse](./okforge_browse.md) — read-only browsing of any OKF bundle.
- [okf_rules](./okf_rules.md) — the OKF v0.1 rules reference behind okforge_browse.
